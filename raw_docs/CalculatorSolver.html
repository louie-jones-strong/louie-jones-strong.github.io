Units Tests

I took this project as an opportunity to learn how to setup unit tests in python.

As there are a large number of puzzles and operations in the game, I found these tests very useful to spot edge case issues.

UX Polishing

I started by creating a terminal program that could solve inputted puzzles.
But I soon expanded it to save and load the puzzles to json, to allow for quick back testing.
I then continue to improve on it to add a graphic user interface, similar to the game, to allow to easier inputting of the puzzles.
